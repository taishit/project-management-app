# 07 リリースノート

## 0.1.0 MVP

### 追加

- プロジェクトCRUD APIを追加。
- 課題CRUD APIを追加。
- ダッシュボード集計APIを追加。
- PostgreSQL向け `management_app.projects` / `management_app.issues` を追加。
- MyBatis Mapper XMLによるDBアクセスを追加。
- Vue 3画面からAPIを呼び出す一覧・登録・編集・詳細画面を追加。
- 課題一覧でプロジェクト、ステータスによる絞り込みを追加。
- 共通エラーハンドリングを追加。

### 変更

- MVPでは認証・認可を対象外とし、`/api/**` はローカル開発で認証なしにした。
- APIパスを `/api` 配下へ統一。
- 旧モック画面を実データ表示へ更新。

### 確認結果

- backend: `mvn test` 成功。
- frontend: `npm.cmd run build` 成功。
