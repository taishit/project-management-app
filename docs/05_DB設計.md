# 05 DB設計

## 想定テーブル

### users

| カラム名 | 型 | 制約 | 説明 |
| --- | --- | --- | --- |
| id | bigint | PK | ユーザーID |
| name | varchar(100) | NOT NULL | 表示名 |
| email | varchar(255) | UNIQUE | ログイン用メール |
| password_hash | varchar(255) | NOT NULL | パスワードハッシュ |
| role | varchar(30) | NOT NULL | ロール |
| created_at | timestamp | NOT NULL | 作成日時 |
| updated_at | timestamp | NOT NULL | 更新日時 |

### projects

| カラム名 | 型 | 制約 | 説明 |
| --- | --- | --- | --- |
| id | bigint | PK | プロジェクトID |
| code | varchar(20) | UNIQUE | プロジェクトコード |
| name | varchar(120) | NOT NULL | プロジェクト名 |
| status | varchar(30) | NOT NULL | 状態 |
| owner_user_id | bigint | FK | 担当ユーザー |
| description | text |  | 説明 |
| created_at | timestamp | NOT NULL | 作成日時 |
| updated_at | timestamp | NOT NULL | 更新日時 |

### issues

| カラム名 | 型 | 制約 | 説明 |
| --- | --- | --- | --- |
| id | bigint | PK | 課題ID |
| project_id | bigint | FK | 所属プロジェクト |
| issue_key | varchar(30) | UNIQUE | 課題キー |
| title | varchar(200) | NOT NULL | 件名 |
| status | varchar(30) | NOT NULL | 状態 |
| priority | varchar(30) | NOT NULL | 優先度 |
| assignee_user_id | bigint | FK | 担当ユーザー |
| due_date | date |  | 期限日 |
| description | text |  | 説明 |
| created_at | timestamp | NOT NULL | 作成日時 |
| updated_at | timestamp | NOT NULL | 更新日時 |
