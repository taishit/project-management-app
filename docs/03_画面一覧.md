# 03 画面一覧

| 画面ID | 画面名 | パス | 概要 |
| --- | --- | --- | --- |
| SCR-001 | ダッシュボード | `/` | 接続状態、件数サマリー、最近更新された課題を表示 |
| SCR-002 | プロジェクト一覧 | `/projects` | プロジェクト一覧、編集、削除、新規作成導線を表示 |
| SCR-003 | プロジェクト登録 | `/projects/new` | プロジェクトキー、名称、説明を登録 |
| SCR-004 | プロジェクト編集 | `/projects/:id/edit` | 既存プロジェクトを編集 |
| SCR-005 | 課題一覧 | `/issues` | 課題一覧、詳細、編集、削除、新規作成導線を表示 |
| SCR-006 | 課題登録 | `/issues/new` | 課題を登録 |
| SCR-007 | 課題詳細 | `/issues/:id` | 課題の詳細項目を表示 |
| SCR-008 | 課題編集 | `/issues/:id/edit` | 既存課題を編集 |

## 表示ラベル

ステータスは `TODO=未着手`, `IN_PROGRESS=対応中`, `REVIEW=レビュー中`, `DONE=完了` と表示する。

優先度は `LOW=低`, `MEDIUM=中`, `HIGH=高` と表示する。
