# 02 決定事項

## 初期方針

- アプリ名称は `ProjectFlow`
- 公開前提のため、社内固有情報や実在企業情報は扱わない
- 画面文言と UI は独自化する
- フロントエンドは Vue 3 + Vite + TypeScript
- バックエンドは Java 21 + Spring Boot + Maven
- DB はローカル PostgreSQL を利用する
- Docker は利用しない

## 実装方針

- 初期フェーズではインメモリデータで API 動作を確認する
- 将来的に repository 層を PostgreSQL 実装へ差し替えやすい構成を採用する
- API は `/api/v1` 配下に集約する
